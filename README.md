# RC Body Scaler

**STL ボディスケーリングツール** — AIで生成した車のSTLデータを、実際のラジコンシャーシに合わせてスケーリングするブラウザ完結ツールです。

ホイールベース・トレッド幅・車高をそれぞれ独立して指定でき、スケール後のSTLをそのままダウンロードして3Dプリントできます。インストール不要、サーバー不要。

---

## 基本的な使い方

右パネルの手順に従って操作します。

| ステップ | 内容 |
|----------|------|
| 01 Load | STLをドロップまたはクリックして読み込む |
| 02 Orient | 回転ボタンでモデルの向きを正しい軸に合わせる |
| 03 Wheels | 前輪・後輪の軸位置をクリックで指定する |
| 04 Dimensions | 目標のホイールベース・車高・トレッド幅をmmで入力する |
| 05 Export | スケール適用 → STLをダウンロード |

---

## 02 / Orientation — 向き合わせ

AIで生成したSTLは向きがバラバラなことが多いため、まずここで正しい向きに合わせます。

**目標の向き:**

- 進行方向（前後）= **X軸（赤）**
- 車高（上下）= **Y軸（緑）**
- 車幅（左右）= **Z軸（青）**

3Dビュー内の軸矢印を見ながら、X/Y/Z の ±90° ボタンで回転を合わせてください。

> ⚠️ 回転すると前後輪の指定はリセットされます。向き合わせを先に完了させてください。

---

## 03 / Wheel Points — ホイール位置指定

ホイールベースの計算基準となる前輪・後輪の軸位置を指定します。

### 手順

1. 「前輪軸をクリック指定」ボタンを押す
   → カメラが自動で**側面ビュー（正投影）**に切り替わる
2. モデル上の前輪タイヤ中心あたりをクリック
3. 矢印キーで位置を微調整
4. 同様に後輪を指定
5. 「✓ 軸位置を確定」ボタン または `Enter` で完了

### キー操作

| キー | 動作 |
|------|------|
| `← →` | X軸方向に 1mm 移動 |
| `↑ ↓` | Y軸方向に 1mm 移動 |
| `Shift` + 矢印 | 0.1mm 単位の微調整 |
| `Tab` | 前輪 ↔ 後輪の切り替え |
| `Enter` | 軸位置を確定してPERSP視点に戻る |
| `Esc` | キャンセル |

### タイヤ半径（可視化用）

タイヤ半径を入力すると指定した軸位置にタイヤ円（リング）が表示されます。実際のタイヤサイズに合わせると位置の確認がしやすくなります。

### Y軸ロック

「前後輪のY軸を同じ高さに固定」にチェックを入れると、片方を上下に動かしたときにもう一方も追従します。地面からの高さを揃えたい場合に便利です。

---

## 04 / Target Dimensions — スケール設定

各軸の目標サイズをmmで入力します。空欄の軸はスケール倍率 1.0（変更なし）になります。

| 項目 | 軸 | 説明 |
|------|----|------|
| ホイールベース | X | 前後輪の中心間距離。例: TT-02 = 257mm |
| 車高 | Y | 地面からボディ最頂部までの高さ |
| トレッド幅 | Z | 左右タイヤ中心間の距離 |

スケールバー（赤=X・緑=Y・青=Z）で倍率をリアルタイムプレビューできます。

---

## 05 / Export — エクスポート

### スケール適用

「▶ スケール適用」を押すと緑色のスケール済みモデルが表示されます。元のモデルは半透明のゴーストとして重ね表示されます。

### ゴースト表示切替

「元モデルをゴースト表示」チェックで元モデルの表示・非表示を切り替えられます。スケール前後の形状比較に使います。

### STLダウンロード

「⬇ STLをダウンロード」で `rc_body_scaled.stl` を保存します。単位は入力値と同じ **mm** です。

---

## 06 / Interior Box — 内部空間確認

ラジコンのシャーシ・バッテリー・受信機などが収まるかを確認するための補助ボックスです。

- チェックを入れると、ホイールベース中心に**黄色のワイヤーフレームボックス**が表示されます
- X（幅）・Y（高さ）・Z（奥行き）を個別に入力してサイズを変更できます
- デフォルト値はホイールベースと車体サイズから**自動計算**されます（スケール適用後は適用後の値を使用）
- 体積（cm³）もリアルタイムで表示されます

## ビュー操作

### 3Dビューの操作

| 操作 | 動作 |
|------|------|
| ドラッグ | 視点を回転 |
| スクロール | ズームイン / アウト |

### PERSP / ORTHO 切替

ヘッダーの `PERSP` ボタンで透視投影と正投影を切り替えられます。

- **PERSP（透視投影）**: 通常の3D表示。奥行き感あり
- **ORTHO（正投影）**: 遠近感なし。寸法確認や軸位置指定に適しています

> 前輪・後輪ボタンを押すと自動でORTHO + 側面ビューに切り替わります。確定後はPERSPに戻ります。

### ダーク / ライト モード切替

ヘッダーの `☀ / 🌙` ボタンでテーマを切り替えます。ライトモードは色弱の方でも識別しやすい配色（青系・橙系）を採用しています。

## よくある質問

**Q. ホイールベースの値はどこで調べる？**
メーカーサイトやシャーシの説明書に記載されています。例: タミヤ TT-02 = 257mm、TA-07 = 257mm、M-07 = 210mm。

**Q. モデルの向きを変えたらタイヤ位置がずれた**
向きを変えると前後輪の指定はリセットされます。02（向き合わせ）を完了させてから03（ホイール指定）を行ってください。

**Q. クリックしてもモデルに当たらない**
正投影（ORTHO）の側面ビューでクリックすると精度が上がります。前輪/後輪ボタンを押すと自動で側面ビューに切り替わります。当たらない場合はXZ平面上の点が使われます。

**Q. ダウンロードしたSTLの単位は？**
入力値と同じ mm 単位です。スライサーソフトに読み込む際は mm を指定してください。
